새 게임 버전과 로더 버전이 일치하는지 확인하고 `설치`를 눌러 업데이트하세요.

:::warning 중요

설치기 실행중인 경우 '프로필 생성'을 반드시 비활성화하세요.

:::

## 3. 마인크래프트 런처에서 프로필 업데이트하기 {#3-update-launcher-profile}

설치가 끝나면, 마인크래프트 런처를 실행하세요. `설치 목록` 탭으로 이동합니다. Fabric 프로필을 선택하고 아래와 같이 수정합니다:

![Minecraft 런처에서 Fabric 로더 버전을 변경하는 모습](/assets/players/updating-fabric.png)

버전을 방금 설치한 Fabric 로더로 선택한 다음, `저장`을 클릭합니다.

## 4. 업데이트 완료 {#4-you-re-done}

위의 과정을 모두 완료했다면 `플레이` 탭으로 돌아가서, 왼쪽 아래의 버전 선택 메뉴에서 Fabric을 선택한 다음 플레이를 클릭하면 됩니다.

설치 과정 중 문제가 발생했다면, [Fabric Discord (영어)](https://discord.fabricmc.net/)의 `#player-support` 채널에서 도움을 요청할 수 있습니다.
